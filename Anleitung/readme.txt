Long Pham

=================== Beschreibung ===============================

‚ToDoLocator‘ ist eine für Applikation, mit der man eine Aufgabe
(ein TODO) mit einer Adresse verknüpft, welche dann auf einer Karte 
markiert ist. Dabei muss der User in jedem Fall eine korrekte Adresse eingeben.

In dem Ordner, in dem sich auch diese Datei befindet, finden Sie eine Bildersammlung, die als Anleitung dient. 

=================== Anleitung ==================================


Schritt 1 :
Nach dem Öffnen werden Sie eine leere Liste sehen, die Sie füllen können,
Indem Sie den Knopf „Second“ auf der unteren Navigationsleiste klicken.

Schritt 2 : 
Nun können Sie eine Aufgabe und eine Adresse in die Textfelder auf der 
oberen Hälfte der Applikation eingeben. 
Beispiel: 
	„Geschwister-Scholl-Straße 1-3“,
	„Maximilian-Kaller-Straße 4“ oder
	„Luxemburger Straße 10“
WICHTIG:
Dabei ist zu beachten, dass man die Adresse nicht beliebig wählen kann.
Das sind die Bedingungen:
- Die Adresse muss es wirklich geben!
- Sie kann auch abgekürzt werden wie in Bild 3 zu sehen (Beispiel 2 & 4)
- Keine Schreibfehler
- Manche Adressen haben eine besondere Syntax wie z.B Luxemburger Straße 10
Die App findet die Straße nur genau unter dieser Bezeichnung, nicht aber unter „Luxemburgerstraße 10“. Bitte beachten!

Haben Sie dies getan, können Sie die TODO mit Adresse über den Knopf 
„add todo“ in die Liste der Aufgaben einfügen.
Nach diesem Prozess können Sie wieder auf die Startseite. Dafür klicken Sie
Auf den unteren Navigationsknopf „First“.

Schritt 3 : 
Nun werden Sie Ihren ersten Eintrag sehen. Oben steht die Aufgabe und direkt
Da drunter steht die Adresse.
Klicken Sie nun auf Ihren Eintrag und Sie werden auf eine weitere
Oberfläche bewegt.

Schritt 4:
Dort werden Sie nochmal Ihre Aufgabe, Adresse sowie Karte mit Markierung ihrer
eingegebenen Adresse sehen.

Schritt 5:
Sie können per Doppel-Klick in die Karte reinzoomen.
